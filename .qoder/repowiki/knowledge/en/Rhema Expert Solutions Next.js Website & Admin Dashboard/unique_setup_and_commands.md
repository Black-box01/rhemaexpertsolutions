Admin password defaults to `rhema2026` if no `rhema_content.admin.password` row exists and is persisted back to the DB on first login. Email requires `SMTP_USER` / `SMTP_PASS` env vars; without them, `sendEmail` logs a warning and returns `{ success: false }` rather than throwing. Development/build/start/lint run via `npm run dev|build|start|lint` as defined in `package.json`.
