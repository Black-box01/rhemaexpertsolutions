- **Framework & Rendering**: Built on Next.js (App Router) with React 19, utilizing Server Components for data fetching in `app/page.tsx` and Client Components (`'use client'`) for interactive UI elements like the `Header` and `AdminDashboard`.
- **Data Layer**: Integrates Supabase as the primary backend. It employs a dual-client strategy: a public client (`lib/supabase.ts`) for read-only operations on the frontend, and an admin client (`lib/supabase-admin.ts`) using a Service Role Key for server-side mutations in `app/actions/`.
- **Action Layer**: Server Actions (`app/actions/*.ts`) encapsulate business logic for authentication, registration submissions, and admin CRUD operations, ensuring secure database interactions behind authentication checks.
- **UI Composition**: Uses a component-based architecture in the `components/` directory for reusable UI elements (e.g., `HeroSlideshow`, `ImageGrid`) and Tailwind CSS v4 for styling, with static assets managed in `public/img/` categorized by service domain.
